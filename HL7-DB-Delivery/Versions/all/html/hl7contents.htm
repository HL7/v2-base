This is a HTML-Version of the specification as maintained by HL7 database. It helps you to browse through the standard quickly. It provides you with navigation elements to gather information about events, messages, segments, tables and table values, (logical) data types, conformance statements (as of v2.4), and last but not least a comparison of data elements among the different HL7 versions (2.1 <=> … <=> v2.8.1).

In the static view, the generated HTML network allows the pursuit of every HL7 point of view you want to. The files generated from the database are hyperlinked among each other, but the original documents are integrated into this HTML network, too. As such, you can access the original segment and field definition. For old versions, i.e. up to and including v2.4, there is no "link back" to the converted standard documents. As a consequence, you have to use the "back button" of your browser. The HTML rendering starting with version 2.5 has been changed, so that it also includes a "back link" as well as full hyperlinking. Both allows to jump directly to all locations, which may be of interest.

Starting with v2.5 the database also contains the standard documentation as well.
