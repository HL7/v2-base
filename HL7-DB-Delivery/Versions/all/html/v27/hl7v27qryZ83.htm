Query Z83: ORU Subscription

Name ORU Subscription
Query-ID Z83
Official Document: 5.7.3.0
Query Type: Publish
Query Mode: Both
Query Message Type: QSB
Query Trigger: Z83
Query Message Struct: QSB_Q16
Query Characteristics: Returns lab results reports for the patient(s) as constrained in the input parameters.
Response Message Type: ORU
Response Trigger: R01
Response Message Struct: ORU_R01
Response Characteristics: A standard query response is not received from the server. Instead, actual ORU messages are returned corresponding to the constraints expressed in the input parameters.
Purpose: Sends Lab Results, either filtered or unfiltered, as specified in the input parameters.
Segment Pattern: R01
